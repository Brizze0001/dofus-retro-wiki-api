# 💻 Exemples — Dofus Rétro API

Exemples prêts à l'emploi pour consommer l'**[API Dofus Rétro](https://wiki.moon-bot.io/api/)** dans plusieurs langages.

| Dossier | Langage | Contenu |
|---------|---------|---------|
| [`bash/`](bash/) | Bash + `curl`/`jq` | Requêtes en ligne de commande |
| [`javascript/`](javascript/) | JavaScript (Node 18+ / navigateur) | `fetch`, liste, détail, calcul de craft |
| [`python/`](python/) | Python 3 | Client `requests`, recherche, recette |
| [`lua/`](lua/) | Lua (LuaSocket/LuaSec) | Récupérer le point faible d'un monstre |

> Base URL : `https://wiki.moon-bot.io/api` — aucune clé, CORS ouvert.
> Merci de **mettre en cache** et de **ne pas marteler** le serveur communautaire.
